Features

No interface. Until you want one.

Flobro windows are pure content: no tabs, no address bar, no chrome. Move your mouse to the top edge and a slim toolbar slides in. Move away and it disappears.

Always on top

Your floating window stays above every app. Toggle it per window with the pin whenever you need normal stacking.

Distraction free

Frameless and silent. The hover toolbar gives you zoom, pin, minimize, settings and close, then gets out of the way.

Actually lightweight

Built with Tauri and Rust on the OS webview. A ~10 MB download that is easy on your memory and battery.

16:9 snap

One click snaps the window to perfect video proportions. Resize freely from any edge at any time.

Per-window zoom

Scale pages from 25% to 500% so a small floating window still shows exactly what you care about.

Open source

MIT licensed. Read the code, file issues, send pull requests.

How it works

Three steps to floating.

01

Paste a link

Open Flobro and drop in any URL: a video, stream, dashboard, docs page, anything on the web.

02

Hit Float

The page opens in a clean frameless window that stays on top of your other apps.

03

Get back to work

Move it, resize it, forget it. Hover the top edge whenever you need the controls.

Chrome extension

Float straight from Chrome.

The Flobro extension pops any tab, link or video out into a minimal window, right from the toolbar or the right-click menu. And with the desktop app installed, one click forwards the page to a true always-on-top window.
