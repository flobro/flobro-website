## [1.0.0] - 2026-07-09

### Added

- Full redesign of flobro.app: animated hero with a draggable, hover-toolbar demo window
- English and Dutch with a language toggle; English inline, other languages fetched as JSON
- Self-hosted subset of the Inter variable font
- Human-friendly privacy page in both languages
- Download buttons resolve the latest installers from the GitHub releases API
- Cookieless PostHog analytics with a lead funnel (download, extension, donate clicks)
- A few easter eggs for the curious
